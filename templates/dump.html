ABOUT ME

Hi, I'm Beatriz Basa, a web developer with a passion for creating functional and intuitive web applications. I’m currently based in Urdaneta City, Pangasinan, and I’m set to graduate with a Bachelor of Science in Information Technology, Major in Web and Mobile Technologies from Pangasinan State University – Urdaneta Campus.


My Journey

I started my journey into the tech world with a curiosity for how the web works and a desire to build solutions that make people's lives easier. Over the course of my academic career, I honed my skills in various programming languages and frameworks, focusing primarily on HTML, CSS, PHP, and SQL. I also explored Dart as I delved into app development, and worked extensively with popular frameworks like Laravel, Bootstrap, and Flutter.

As a web developer, I’ve been privileged to work on a variety of projects, each presenting unique challenges and learning experiences. My work at the PNP Urdaneta City Police Station during my internship gave me the opportunity to lead the development of two critical systems: the WCPD & Investigation Information Management System and the Admin File Management System. These projects taught me not only the technical skills needed to build complex applications but also how to collaborate effectively with different departments and stakeholders.


What I Do

In the world of web development, I specialize in full-stack development using Laravel and MySQL, where I build, maintain, and optimize web applications. I focus on:

  • Developing intuitive user interfaces using HTML, CSS, and responsive frameworks like Bootstrap

  • Building and integrating relational databases to handle large amounts of data with MySQL and phpMyAdmin

  • Implementing core functionalities such as CRUD operations (Create, Read, Update, Delete) to make managing and organizing information seamless

  • Utilizing Git for version control, ensuring smooth collaboration and project management



Projects That Define My Work

WCPD & Investigation Information Management System

As the main developer, I led the creation of a web-based information management system for the PNP Urdaneta City Police Station. This system streamlined the process for investigators by providing an efficient way to manage and organize complaint forms. I developed crucial features, such as CRUD functionalities, and even integrated a feature for downloading forms as PDFs using Laravel's DomPDF package.


Admin File Management System

I directed the development of this system to help administrative officers easily monitor and upload police personnel files. Using Laravel and MySQL, I implemented a solution that not only made file management more accessible but also helped the station improve its internal workflow.


QualiFish – Capstone Project

For my capstone project, I took the lead in developing QualiFish, an inventory management system designed specifically for fish farmers and fish farm owners. The system includes prescriptive forecasting features, enabling users to predict inventory needs. I integrated SQL queries with Eloquent ORM and utilized Google Charts to create a user-friendly dashboard for visualizing data.


Looking Ahead

As I approach the completion of my degree, I’m excited about the future and eager to take on new challenges in the tech industry. I’m particularly interested in opportunities that allow me to continue improving my skills in web and software development, and I am always looking to work on projects that have a real-world impact.

I am passionate about learning, solving problems, and delivering high-quality, efficient solutions. I believe that with my technical skills and experience, I can contribute significantly to any team I join, and I’m excited to see where this journey takes me.
